 テーマ「bootstrap2015」2015/07/10版
====================================

------------------------------------
 テーマフォルダ直下のテンプレート
------------------------------------
■ 404.html
404ページで表示されるテンプレートです。

■ calendar.html
カレンダーモジュールで次のページに送ったときのテンプレートです。

■ index.html
一覧ページのテンプレートです。

■ javascript_sample.html
Bootstrapで配布されているjavascriptをあらかじめ実装しているモジュールをまとめたテンプレートです。

■ layout.html
レイアウト機能が実装されたテンプレートです。

■ template.yaml
bootstrap2015テーマを選択したときに、テンプレートをまとめて設定できるファイルです。使用するかどうかは管理画面＞コンフィグ＞テーマ設定 にて変更できます。

■ top.html
トップページのテンプレートです。


------------------------------------
フォルダ構成
------------------------------------
■ admin
bootstrap2015テーマオリジナルの管理画面のテンプレートが入ったフォルダ。
bootstrapテーマでは主に以下のテンプレートが入っています。
・ カスタムフィールド（ブログ・カテゴリー・エントリー・モジュール）
・ レイアウト機能

■ css
bootstrap2015テーマで使っているCSSが入ったフォルダです。

■ fonts
Bootstrapで配布されいているシンボルフォントが入ったフォルダです。

■ images
bootstrap2015テーマで使っている画像が入ったフォルダです。

■ include
bootstrap2015テーマ内のインクルードしているテンプレートをまとめたフォルダです。

■ js
bootstrap2015テーマで使っているJavascriptが入ったフォルダです。

■ less
bootstrap2015テーマで使っているLESSファイルが入っているフォルダです。このlessフォルダに入っているLESSファイルをコンパイルしてCSSを出力しています。

 a-blog cmsの基本機能
------------------------------------
■ カスタムフィールド

ブログ
・ サイトの設定
・ SEOの設定
・ OGPの設定
・ ブログの設定
・ ヒーローの設定

カテゴリー
・ SEOの設定
・ OGPの設定

モジュール

・ MID別の設定


------------------------------------
 カスタマイズについて
------------------------------------
■ スクロールイベントを取得するJS
スクロールイベントを取得するために、Waypointsというプラグインを使用しています。使用方法に着いてはWaypointsの公式サイトをご覧ください。
　・ Waypoints http://imakewebthings.com/waypoints/

■ レイアウト機能
レイアウト機能は http://ドメイン名/bootstrap2015/layout.html にアクセスしてご利用できます。レイアウト機能をトップページでご利用になるときはlayout.htmlをtop.htmlにリネームしてください。
